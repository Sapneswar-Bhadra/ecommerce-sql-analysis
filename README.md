# E-commerce SQL Analysis Project

## Project Overview
This project focuses on analyzing e-commerce data using SQL (MySQL).
The dataset includes orders, customers, delivery, distance, and return analysis.

## Tools Used
- MySQL
- MySQL Workbench
- SQL (Joins, Group By, Case, Window Functions)

## Key Analysis Performed
- Customer return rate analysis (state-wise, customer-wise)
- COD vs Prepaid order behavior
- Delivery delay and distance impact
- High-risk customer identification
- KPI metrics using SQL window functions
